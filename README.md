
# hp-official-website-web

HP 产品官网【Powehi】

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```

## 测试环境打包

$ npm run buildtest
